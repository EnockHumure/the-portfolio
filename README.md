
# Personal Portfolio Website

A modern, responsive, and clean personal portfolio website showcasing my work, skills, vlogs, 
and journey as a software engineer. This portfolio is designed to be simple, professional,
easy to navigate, and fully optimized for all screen sizes.

---

## 🚀 Live Overview

This portfolio includes the following sections:

### 1. Home / Hero Section
A fullscreen introduction with my name, professional title, and a strong call-to-action button inviting users to explore more about me.

### 2. About Me
A clear and friendly section describing who I am, my passion for technology, and my journey as a self-taught web developer and future machine-learning engineer.

### 3. Projects Section
A collection of my featured projects displayed in a clean grid layout. Each project comes with:

- A title
- A short description
- A “View More” button that opens a dedicated page for full project details

The entire layout is user-friendly, organized, and designed for easy navigation.

### 4. Vlogs Section
A unique section showcasing some of my daily experiences as a software engineer. It includes 6 vlog cards, each centered, with full padding, subtitles, and professional presentation.

**Sample vlog themes:**

- Day at kLab
- Coding challenges
- Hackathon experience
- Day in the life of a developer
- Teamwork moments
- Study and coding sessions

This makes my portfolio more personal, engaging, and relatable.

### 5. Contact Section
A simple contact form and icons that make it easy for anyone to reach out to me for collaboration or inquiries.
